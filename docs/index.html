Política de Privacidad

Última actualización: 3 de agosto de 2025

1. No recopilación de datos

La aplicación “Catálogo de Películas” no recopila información personal ni técnica de los usuarios. Esto incluye:

  • No se solicita nombre, correo electrónico ni ubicación.
  • No se accede a funciones del dispositivo como cámara, contactos, almacenamiento o GPS.
  • No se emplean cookies ni tecnologías de rastreo.
  • No se registran estadísticas de uso ni identificadores únicos.

2. Acceso a Internet

La aplicación se conecta a Internet únicamente para obtener datos públicos de películas a través de una API externa (The Movie Database – TMDb). No se transmite información del usuario ni del dispositivo.

3. Privacidad de menores

La aplicación es segura para usuarios de todas las edades y no está dirigida específicamente a menores.

4. Cambios a esta política

Si en futuras versiones se integran funciones que recopilen información, esta política será actualizada y los usuarios serán notificados oportunamente.
